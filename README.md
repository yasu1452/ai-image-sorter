# ai-image-sorter
AI画像をプロンプト・メタデータで仕分けする非公式ツール

https://github.com/yasu1452/ai-image-sorter/releases

## 主な機能

- 画像メタデータ（生成プロンプト等）を利用した自動仕分け
- PNG / JPG / JPEG / WEBP 対応
- 最初に一致したキーワードのみを適用（優先度制）
- 表記ゆれ吸収（スペース / _ / - を同一扱い）
- AND / OR 条件指定対応
- グループ単位での仕分け実行
- 移動 / コピー切り替え
- Undo（直前の仕分けを元に戻す）

## キーワードの書き方

- `A & B` ：AND（両方含む）
- `A | B` ：OR（どちらか含む）

表記ゆれは自動的に吸収されます。

例：
- kind smile
- kind_smile
- kind-smile

上記はすべて同一キーワードとして扱われます。

## 注意

- 本ツールは個人開発による非公式ツールです
- 特定の生成サービス・ソフトとは公式な関係はありません
- 画像の著作権や利用条件は各自でご確認ください
